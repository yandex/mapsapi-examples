Анимировать путь между точками можно с помощью изменения длины линии. Для этого необходимо добавлять новые точки, не меняя всю линию целиком.

Для того, чтобы добавить новые точки в геометрию ломанной линии можно использовать метод set, который принимает на вход индекс точки(которую надо поменять) и новые координаты.

Также геометрию ломанной линии можно поменять целиком с помощью метода setCoordinates.
