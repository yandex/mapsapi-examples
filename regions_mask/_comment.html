Пример показывает, как отобразить на карте только те тайлы, в которые попадает территория нужной страны.

Чтобы залить всю карту белым цветом, можно создать один многоугольник, покрывающий весь мир. У многоугольника можно нарисовать вложенный контур, который будет образовывать полость. Через неё будет видно нужный регион.
