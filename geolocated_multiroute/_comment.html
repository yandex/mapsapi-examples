Данный пример демонстрирует, как добавить кнопку для построения маршрутов на карту и задать начальную точку маршрута с помощью геолокации.

Для определения местоположения используется метод get функции geolocation. В его параметрах можно указать предпочтительный способ определения местоположения (по IP или с помощью Geolocation API), время ожидания ответа и др. Подробнее см. в описании метода.

При вызове данного метода браузер пытается определить географическое положение пользователя. В случае успеха результат подставляется в поле адреса пункта отправления.
