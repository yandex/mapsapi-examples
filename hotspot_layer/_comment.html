Для добавления на карту большого числа интерактивных объектов используется технология активных областей.

Изображения объектов добавляются на карту с помощью «картиночного слоя». Для его создания используется класс Layer.

Информация об объектах (их границах и позиции) хранится на сервере и добавляется на карту через слой активных областей. Для создания этого слоя используется класс hotspot.Layer.

Перед созданием слоя активных областей необходимо создать его источник данных — объект, который будет запрашивать с сервера данные активных областей. Источник данных реализуется классом hotspot.ObjectSource.
