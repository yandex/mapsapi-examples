There are two ways to create a placemark: using the GeoObject class with the "point" geometry type, or using the Placemark auxiliary class.

When creating a placemark, you need to specify the coordinates of the corresponding point. You can also set the placemark's properties (such as the icon text) and options (such as the icon image).

Placemarks can be grouped into collections, as well as clusters (see the examples Collection of geo objects and Placemark cluster).
