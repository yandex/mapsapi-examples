Rules for displaying routes are set in the router options.

The following prefixes are available for setting options for sections of a multiroute:

  • wayPoint - display options for waypoints.
  • wayPointStart - display options for the starting waypoint.
  • wayPointFinish - display options for the destination waypoint.
  • viaPoint - display options for thru points.
  • pin - display options for point markers at waypoints.
  • route - display options for routes.
  • routeActive - display options for the active route.
  • routePedestrianSegment - options for lines of pedestrian route segments.
  • editor - multiroute editor options (see ).
