Многоугольник, прямоугольник и круг можно залить фоновым изображением. Для этого нужно указать ссылку на фоновое изображение в опции fillImageHref.

Тип заливки фоном - fillMethod. Опция может принимать одно из двух значений:

  • stretch - фоновое изображение растягивается по размеру оверлея;
  • tile - фоновое изображение повторяется без изменений размера.
