Редактор геометрии "Прямоугольник" на данный момент не реализован в JS API Карт. Но визуальный редактор доступен для многоугольников. Ссылка на редактор содержится в поле editor объекта-многоугольника. Данный пример показывает, как можно эмулировать редактор прямоугольника через редактор многоугольника.
