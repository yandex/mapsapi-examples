YMapsML - язык описания географических данных. С помощью YMapsML можно настраивать внешний вид географических объектов, отображаемых на карте, а также задавать параметры самой карты.

Для задания параметров карты используется YMapsML-элемент repr:View. С его помощью можно задать тип и область показа карты. В данном примере карте задается тип "спутник".
