Менеджер для расчета оптимальных отступов от краев контейнера карты.

У каждой карты есть собственный менеджер отступов. Чтобы добавить область над картой в менеджер, используйте метод addArea Метод addArea возвращает объект (аксессор), представляющий занятую прямоугольную область. Аксессор позволяет изменить, либо удалить область из менеджера объектов.

Область описывается в виде объекта, который содержит информацию о смещении от краев карты и размер. В качестве единиц измерения возможно использовать пиксели (px), проценты (%). Проценты вычисляются относительно размеров контейнера с картой.

Чтобы добавить область, занимаемую контролом, в менеджер объектов, при создании выставите опцию adjustMapMargin в значение true.

, поддерживают опцию useMapMargin, которая позволяет учитывать отступы карты.
Эту опцию поддерживает ряд методов карты, например setBounds, panTo, setCenter.
