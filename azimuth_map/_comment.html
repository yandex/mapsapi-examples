API Яндекс.Карт может использоваться для отображения на сайте карт разных картографических проекций, сохранив привязку к географическим координатам.

Для того чтобы узнать подробнее, как создать собственную карту, воспользуйтесь примером Создание собственной карты.

Чтобы сохранить привязку к географическим координатам, нужно самостоятельно реализовать методы интерфейса проекции.
